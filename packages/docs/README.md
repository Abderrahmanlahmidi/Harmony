# Harmony Documentation

This is the documentation website for Harmony React UI.

## Development

```bash
# Install dependencies (from root)
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Structure

- `src/pages/` - Documentation pages
- `src/components/` - Documentation-specific components (not part of the library)
- `src/router/` - Routing configuration
- `src/constants/` - Constants and data

The documentation imports the component library from `harmony-react-ui` workspace package.

# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release of @harmony-ui/react
- **Form Components**:
  - Button with variants (primary, secondary, outline) and sizes
  - ButtonIcon for icon-only actions
  - Input with validation states and icons
  - Checkbox with single and group support
  - Radio with single and group support
  - Toggle switches with smooth animations
  - SearchBar with loading states and shortcuts
  
- **Feedback Components**:
  - Alert with variants (info, success, warning, danger)
  - Toast notifications with auto-dismiss
  - Modal dialogs with semantic variants
  - Progress bars with multiple variants and sizes
  - Spinner with customizable sizes and colors
  
- **Display Components**:
  - Card with header, content, and footer sections
  - Badge with multiple variants and sizes
  - Typography component for consistent text styling
  
- **Navigation Components**:
  - ProfileDropdown with user menu

- **Features**:
  - Full TypeScript support with comprehensive type definitions
  - Accessible components following WCAG 2.1 guidelines
  - Smooth animations powered by Framer Motion
  - Dark mode support
  - Responsive design
  - Tree-shakeable exports
  - Storybook documentation for all components

### Technical
- Built with TypeScript 5.9
- Styled with Tailwind CSS 4.x
- Animated with Framer Motion 12.x
- Icons from Lucide React
- Bundled with tsup for optimal output
